# React + Vite Project

Este proyecto es una aplicación web creada con React y Vite. Vite es una herramienta de construcción de proyectos que proporciona un entorno de desarrollo rápido y optimizado para aplicaciones de React.

## Requisitos Previos

Antes de comenzar, asegúrate de tener instalados los siguientes programas en tu sistema:

- [Node.js](https://nodejs.org/) (versión 14 o superior)
- [npm](https://www.npmjs.com/) o [Yarn](https://yarnpkg.com/) (gestor de paquetes)

## Instalación

1. **Clona el repositorio**

   En git bash clona el repositorio con:

   git clone https://github.com/Anima-Tec/2024_Proyecto_Integrador_FRONTEND_Equipo_B.git

   Luego utiliza:

   cd 2024_Proyecto_Integrador_FRONTEND_Equipo_B

   Y por ultimo:

  - npm install

        O

  - yarn install

  Y luego:

  - npm run dev

        O
  
  - yarn dev

  ## Posibles problemas

  - Si tienes problemas para iniciar la aplicacion que con los pasos anteriores no deberia, instala las siguientes dependencias

  - react - Biblioteca principal de React.
  - react-dom - Dependencia para interactuar con el DOM.
  - vite - Herramienta de construcción y desarrollo.
  - @vitejs/plugin-react - Plugin para soportar React en Vite.

  - npm install nombre-dependencia

                  O

  - yarn add nombre-dependencia
